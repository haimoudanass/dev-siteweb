# dev-siteweb

Application **Taskflow** — tableau de bord de gestion de projets (React + Vite + TypeScript).

## Démarrage

```bash
npm install
npm run dev:full
```

- **Frontend** : [http://localhost:5173](http://localhost:5173) (Vite)
- **API mock** : [http://localhost:4000](http://localhost:4000) (`json-server` sur `db.json`)

## Scripts

| Commande      | Description                    |
|---------------|--------------------------------|
| `npm run dev` | Frontend seul                  |
| `npm run api` | API JSON seule                 |
| `npm run dev:full` | Frontend + API en parallèle |
| `npm run build`    | Build production            |
